如果節點是採用VMware Workstation Pro的虛擬機，除了Guest OS要參考 https://yuanrui919.github.io/autorun/ 設定以外，Host OS同樣也要取消帳戶密碼，讓它能自動登入，並且還要啟動虛擬機。

 

針對每一個虛擬機都編寫一個批次檔，例如

l   start_Router.bat：

"C:\Program Files (x86)\VMware\VMware Workstation\vmware.exe" -t -x C:\VM\Router\Router.vmx

exit

 

l   start_Yuanrui.bat：

"C:\Program Files (x86)\VMware\VMware Workstation\vmware.exe" -t -x C:\VM\Pi_Node_Yuanrui\Pi_Node_Yuanrui.vmx

exit

 

l   start_Grace.bat

"C:\Program Files (x86)\VMware\VMware Workstation\vmware.exe" -t -x C:\VM\Pi_Node_Grace\Pi_Node_Grace.vmx

exit

 

再編寫另一個批次檔start_vm.bat，依序呼叫先前的批次檔，啟動虛擬機（請先啟動單線多撥的VM）

l   start_vm.bat：

start C:\VM\start_Router.bat

timeout /t 60

 

start C:\VM\start_Yuanrui.bat

timeout /t 60

 

start C:\VM\start_Grace.bat

exit

 

將start_vm.bat放在C:\Users\使用者帳戶\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
