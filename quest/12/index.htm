支付

將一定數量的特定資產發送到目標賬戶

 

任務連結

https://quest.stellar.org/learn/series/1/quest/2

 

1.      按「FUND」建立任務帳戶

 

2.      前往 Stellar Laboratory 並在主導覽列中選擇 Build Transaction 頁面。

 

3.      Source Account輸入任務帳戶公鑰，並取得交易序列號

 

4.      Operation Type選「Payment」。底下還有幾個欄位

l   Destination是接收的帳戶，你可以重新建一個新帳號，或者像我用上一個任務的Account 2。

l   Asset是要發送什麼資產。對於這個任務，我們將使用原生(native) XLM 代幣，關於自定義資產將在稍後介紹。

l   Amount是要發送多少資產。就跟現實銀行轉帳一樣，你有多少資產才能發送多少給別人，而且還要算上手續費。

 

5.      簽署交易

 

6.      Add Signer輸入任務帳戶的私鑰

 

7.      交易簽署後，繼續提交

 

 

8.      如果交易成功，回到 Stellar Quest Learn 頁面上進行驗證並領取您的新徽章。
