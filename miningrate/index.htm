挖礦速率

 

點擊Pi APP首頁的閃電，可以看到詳細的挖礦速率

 

除了可看到挖礦週期倒數結束的時間(還有多久要點閃電)，還有挖礦速率公式：

總挖礦速率 = 基礎速率 * 加速器 * 獎勵

 

l   基礎速率

每個月會根據已被開採的Pi數量，以及先鋒人數來調整，以控制Pi的產出，維持稀缺性。原則上，速率將會越來越低。

l   加速器

加速器 = 先鋒 + 安全圈 + 鎖倉獎勵

n   先鋒

表示你要主動挖掘，每天要點閃電，不然挖礦是不會進行的。它是1(100%)，是為了在沒有任何其他獎勵的情況下，使乘數保持1，讓公式不會乘起來變成0。

n   安全圈

要等挖Pi三次之後才能設定，從通訊錄加入你信任的人，只要5位就能加到100%。詳細請參考 https://yuanrui919.github.io/securitycircle/ 。

n   鎖倉獎勵

可自由選擇鎖倉的比例與時間。類似定存的概念：雖然Pi幣是你的，但鎖倉期間是不能使用的。但獎勵不是直接發利息給你，而是提高挖礦速率。詳細請參考 https://yuanrui919.github.io/lockup/ 。

l   獎勵 = 先鋒 + 推廣團隊 + 應用使用獎勵 + 節點獎勵

n   先鋒

同前面的解釋。

n   推廣團隊

包含「邀請你的人」，以及「你邀請的人」。要注意，這些人必須是處於"在線"的狀態才有獎勵，所以請互相提醒團隊的人點閃電挖Pi。

n   應用使用獎勵

使用Pi相關的APP，例如聊天室、區塊鏈瀏覽器、錢包，或是逛逛Pi Browser內的頭腦風暴(brainstorm)。

n   節點獎勵

Pi Network區塊鏈是去中心化的系統，交易的驗證、帳本的儲存都仰賴全球成千上萬的節點運行(手機挖礦只是行銷話術)，如果有閒置的電腦(文書機，8GB記憶體，跑得動Win10就可以)，建議能安裝節點程式。詳細請參考 https://yuanrui919.github.io/installnode/ 。

 

調降挖礦速率是一件好事，控制Pi幣的稀缺性，將有助於Pi幣價值的提升。而且能維持Pi生態的長期發展，希望我們能在十幾、二十年後還繼續挖Pi，而不是在這一兩年內就挖完了。將來也許每天能挖到的Pi很少，但Pi卻很有價值，能吸引源源不斷的人潮加入挖Pi，Pi的生態才能走得更健全、更長遠。

 

所以請記得每天準時點閃電，不要讓時間浪費掉囉。
